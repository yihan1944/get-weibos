# 饭否美好时光

Chrome 浏览器插件，用于将饭否推文下载为 Markdown 或 HTML 格式（可打印为 PDF）。

## 功能特性

1. **下载全部推文** - 一键导出所有推文
2. **下载指定数量推文** - 最近 n 条，n 为正整数
3. **自动翻页** - 支持翻页获取大量推文，显示实时进度
4. **内容格式** - 包含发送时间和推文内容（图片推文显示"图片"）
5. **按日期分组** - 推文按日期自动分组
6. **支持 Markdown / HTML** - 可选 Markdown 或 HTML 格式导出
7. **Chrome 浏览器插件** - 安装后基于用户本地电脑运行

## 安装方法

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"按钮
5. 选择本项目文件夹
6. 插件图标将出现在浏览器工具栏

## 使用方法

1. 打开饭否网站 https://fanfou.com 并登录
2. 点击浏览器工具栏中的插件图标
3. 选择下载范围（全部 / 最近 n 条）
4. 选择输出格式（Markdown / HTML）
5. 点击"下载推文"按钮
6. 选择保存位置

## 输出格式

### Markdown

按日期分组，使用引用块展示推文内容，支持 emoji 标记。

```markdown
# 📝 饭否美好时光

> 📅 导出时间：2026/06/26 10:00
> 📊 共 20 条推文

---

**2026-06-26**

_10:00_

今天天气真好！

---
```

### HTML

排版美观的 HTML 页面，可直接在浏览器中打开查看，按 `Ctrl+P` 即可打印为 PDF。

## 注意事项

- 需要先登录饭否才能使用
- 大量推文导出需要逐页获取，请耐心等待
- 插件运行时不会改变当前页面

## 开发说明

- 使用 Chrome Extension Manifest V3
- 通过 `chrome.scripting.executeScript` 注入脚本获取推文数据
- 使用 `fetch` API 获取分页数据，避免页面跳转
- 推文选择器：`#stream span.content` / `#stream a.time`
