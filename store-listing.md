# Chrome 网上应用店 上架信息

## 商品名称
饭否美好时光

## 简短描述（132 字符以内）
将饭否推文下载为 Markdown 或 HTML 格式，支持全部或指定数量导出，自动翻页获取。

## 详细描述
饭否美好时光是一款 Chrome 浏览器插件，帮助你备份和保存饭否上的推文。

功能特点：
• 下载全部推文或最近 N 条推文
• 自动翻页获取大量推文，显示实时进度
• 按日期分组展示推文
• 支持导出为 Markdown 格式，排版美观
• 支持导出为 HTML 格式，可直接打印为 PDF
• 包含发送时间和推文内容
• 图片推文自动标记

使用方法：
1. 安装插件后，打开饭否网站并登录
2. 点击浏览器工具栏中的插件图标
3. 选择下载范围和输出格式
4. 点击「下载推文」按钮

所有数据处理均在本地完成，不会上传任何信息到第三方服务器。
